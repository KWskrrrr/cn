# 产品概述


H5页面检测系统是基于页面错误、性能、安全、AI等维度，针对所有H5页面进行代码检测，第一时间跟踪有问题的页面，对问题进行快速定位分析。

## 主要特性

* 检测涵盖了异常、性能、合规、安全等维度，确保页面检测的广度和深度。
* 每日定时发送检测报告，检测结果会以图表的形式进行展示，定时排查关注的页面是否存在异常。
* 支持多页面的同时检测，提高检测效率。
* 支持用户自定义检测指标的阈值，配置满足条件的指标项。
