# 产品功能

京东云网格提供了如下功能：

- 部署/升级/卸载网格

基于Kubernetes集群提供网格的一键安装部署、升级网格版本、卸载网格。

- 网格组件管理

用户可设置网格的配置组件的开关。

- 服务网关（Ingress Gateway）

配置服务对外提供的网关信息。

- 虚拟服务（Virtual Service）

配置服务的内部/外部分流规则。
